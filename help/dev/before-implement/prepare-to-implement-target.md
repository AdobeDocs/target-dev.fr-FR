# Préparation à l’implémentation de [!DNL Target]

Tâches à effectuer et informations à collecter lors de la préparation de l’implémentation de [!DNL Adobe Target].

Le respect de ces instructions et le remplissage des questionnaires fournis vous aideront à implémenter [!DNL Target], que vous l’implémentiez vous-même ou que vous travailliez avec [!DNL Adobe Consulting] (recommandé).

Si vous travaillez avec un consultant Adobe, il n’est pas nécessaire de connaître les informations contenues dans ces documents avant d’implémenter [!DNL Target]. Cependant, le processus préalable à la mise en œuvre peut être plus rapide si vous connaissez déjà les documents créés et les questions posées par Consulting. Il se peut que ces documents soient modifiés à l’avenir.

## [!DNL Adobe Target] demande d’informations de l’entreprise

Cette section contient le document de demande d’information (RFI) de l’entreprise que le client doit remplir avant l’appel de lancement de la mise en œuvre de l’[!DNL Target].

Ce document permet ce qui suit :

* Comprendre les objectifs commerciaux du client
* Comprendre les objectifs d’optimisation du client

Téléchargez le document [Business Request for Information](assets/business-rfi.docx).

## [!DNL Adobe Target] demande d&#39;information technique

Cette section contient le document de demande d&#39;information technique (DDR) qui aidera notre équipe de conseillers à développer une implémentation [!DNL Target] qui répondra aux exigences commerciales et techniques du client. Les questions posées dans ce document visent à identifier les difficultés ou obstacles sous-jacents afin de réduire, voire d’éliminer, la nécessité de modifier la mise en œuvre.

Ce document permet ce qui suit :

* Comprendre la plateforme technique et ses éventuelles limitations
* Comprendre l’environnement technique en corrélation avec les personnes et les processus

Téléchargez le document [Demande technique d’informations](assets/technical-rfi.docx).
