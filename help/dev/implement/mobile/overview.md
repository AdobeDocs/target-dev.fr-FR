# [!DNL Target] pour les applications mobiles

[!DNL Adobe Target] permet de tester, de personnaliser et d’optimiser les expériences d’applications mobiles en fonction du comportement des utilisateurs et du contexte mobile. Vous pouvez diffuser des interactions qui entrent en contact et convertissent aussi bien par le biais de tests itératifs, que d’une personnalisation basée sur les règles et sur l’intelligence artificielle.

## [!DNL Adobe Experience Platform Mobile SDK]

Le [!DNL Adobe Experience Platform Mobile SDK] est la solution recommandée pour alimenter les solutions et services [!DNL Adobe Experience Cloud] dans vos applications mobiles.

Pour plus d’informations sur l’implémentation de [!DNL Target] sur les applications mobiles, consultez la documentation de [Adobe Experience Platform SDK pour applications mobiles](https://developer.adobe.com/client-sdks/documentation/){target=_blank}. Vous trouverez des informations spécifiques à [!DNL Target] sous [Adobe Target](https://developer.adobe.com/client-sdks/documentation/adobe-target/){target=_blank}.

Découvrez comment implémenter des applications [!DNL Adobe Experience Cloud] dans votre application mobile à l’aide de l’[!DNL Adobe Experience Platform Mobile SDK]. Pour plus d’informations, consultez le tutoriel [Implémentation de Adobe Experience Cloud dans les applications mobiles](https://experienceleague.adobe.com/docs/platform-learn/implement-mobile-sdk/overview.html?lang=fr){target=_blank} dans le tutoriel [!DNL Experience Platform] *Tutoriel sur les applications mobiles*.
